HTML

  • The head element contains information about the webpage.
  • The meta element, found in the head, stores information useful for the browser.
  • The body element represents the visible content shown to the user.
  • The header element formats text into headers into tiers (i.e. h1-h6).
  • The u, o, and l elements create lists; u makes unorganized lists, o makes organized lists, and l organizes the elements in those lists.
  • The section element organizes text into sections and is formated using classes and IDs.
  • The p element organizes text into paragraphs.
  • The br element makes a line break and adds a new blank line.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout branch-name: takes user to the specified branch.
  • cd "file location": navigates git into the specified file location.
  • cd .. : navigates git out one layer from the current file location on the file path.
